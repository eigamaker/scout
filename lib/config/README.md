# デバッグ設定

このディレクトリには、開発・デバッグ用の設定ファイルが含まれています。

## ファイル構成

- `debug_config.dart` - デバッグモードの設定管理

## デバッグ機能

### デバッグモード
- メインメニューでデバッグモードを切り替え可能
- デバッグモード有効時、選手詳細画面にデバッグボタンが表示される

### デバッグ画面
- 真の能力値と表示能力値の比較
- スカウトスキルによる精度計算の詳細表示
- 技術面・メンタル面・フィジカル面の能力値比較

### デバッグ表示
- 選手詳細画面で真の能力値を括弧内に表示
- 精度計算の詳細情報を表示

## 使用方法

1. メインメニューで「デバッグモード」ボタンを押してデバッグモードを有効化
2. 選手詳細画面で虫のアイコンを押してデバッグ画面を開く
3. スカウトスキルを調整して精度の変化を確認
4. 真の能力値と表示能力値の差を確認

## 注意事項

- デバッグ機能は開発・テスト用です
- 本番環境では無効にすることを推奨します
- デバッグ情報はゲームの没入感を損なう可能性があります
